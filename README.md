# CREDI FLUX
CASH FLOW MINIMIZER SYSTEM FOR OPTIMIZING DEBT SETTLEMENT AND FINANCIAL 
TRANSACTION MANAGEMENT 

# Description
In many situations, a group of individuals borrow money from each other, which creates 
multiple financial transactions. Managing these transactions manually becomes complex 
and inefficient. Settling debts individually may result in unnecessary payments and 
increased cash movement. 

This project solves this problem by developing a Cash Flow Minimizer system that 
reduces the total number of transactions required to settle debts. It uses graph 
algorithms along with optimization techniques such as the Min-Cost Max-Flow algorithm 
to ensure debts are settled in minimum transactions.
